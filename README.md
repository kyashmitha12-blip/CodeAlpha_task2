# AI Chatbot

## Description
This is a rule-based AI chatbot designed to respond instantly to user queries
using predefined input patterns.

## Features
- Instant responses
- Predefined patterns
- Lightweight and fast

## Technologies Used
- Python
- JSON

## How to Run
1. Run `python3 chatbot.py`
2. Type a message
3. Get instant response
